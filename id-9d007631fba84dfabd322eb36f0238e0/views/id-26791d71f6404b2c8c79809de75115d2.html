02 Infrastructure Usage Diagram ()
Infrastructure usage diagram atau diagram penggunaan infrastruktur bertujuan untuk menganalisis kinerja dan skalabilitas karena berhubungan dengan infrastruktur fisik yang digunakan untuk mendukung aplikasi. Informasi yang terdapat dalam diagram ini berupa bagaimana aplikasi didukung oleh infrastruktur software dan hardware layanan infrastruktur (infrastructure service) yang diberikan untuk aplikasi oleh device, software sistem, dan network. Diagram penggunaan infrastruktur yang telah kami buat terdiri dari elemen Application Component, Technology Service, Node, dan System Software.
