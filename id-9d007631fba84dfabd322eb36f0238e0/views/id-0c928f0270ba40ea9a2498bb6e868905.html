02 Logical Data Diagram ()
Logical data diagram atau diagram data logikal bertujuan untuk memperlihatkan pandangan logikal terhadap relasi/hubungan antar entitas dan menyiapkan rancangan basis data atau database. Informasi yang terdapat dalam diagram ini berupa arsitektur aplikasi, diagram entitas, dan skema database. Diagram data logikal yang telah kami buat terdiri dari elemen Data Object.
