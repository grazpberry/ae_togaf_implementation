04 Application Structure Diagram ()
Application structure diagram atau diagram struktur aplikasi bertujuan untuk merancang atau memahami struktur utama aplikasi atau komponennya beserta data yang berhubungan dengannya. Informasi yang terdapat dalam diagram ini berupa struktur satu atau lebih aplikasi atau komponen aplikasi. Diagram struktur aplikasi yang telah kami buat terdiri dari elemen Application Interface, Application Component, dan Data Object.
